◆ openPar()

ConstraintBuilder& vara::feature::ConstraintBuilder::openPar ( )
inline

Open parenthesis.

Allows to add more precise parentheses.

Example: feature("A").implies().feature("B").implies() .openPar().feature("C").implies().feature("D").closePar() produces the constraint '((A => B) => (C => D))'

Example: lNot().openPar().lNot().feature("A").implies().feature("B").closePar() produces the constraint '!(!A => B)'
